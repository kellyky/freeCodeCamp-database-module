# freeCodeCamp-database-module
Here you'll find mostly exercises from the freeCodeCamp Relational Database course (currently in beta, but it's awesome and there's a lot of bash). 

I also ended up rewriting a [text adventure game I coded in python for a Udacity Course](https://github.com/kellyky/Adventure-Game-Python) a while back. 😆 Credit to Udacity (Intro to Programming) for the plot; the code is my own, as are the villains, and most of the weapons. 

To play, download/clone the text-adventure.sh. In your terminal, navigate to the directory it's in and give it executable permissions: `chmod +x text-adventure.sh`. Then to play it run it `./text-adventure.sh`. Enjoy!

The same applies to running the other .sh files if you're curious.
